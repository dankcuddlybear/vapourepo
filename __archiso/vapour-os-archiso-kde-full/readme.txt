Vapour OS x86_64 KDE
KDE Plasma is a beatiful, advanced desktop environment with a familiar feel.
This edition of Vapour OS features a customised KDE Plasma desktop session,
with pretty colours, wallpapers and icons.

Minimum system requirements:
CPU: 1GHz
GPU: With OpenGL/Vulkan support
RAM: 1.6GiB (5GiB to boot from RAM)
Storage: 10GiB

Recommended system requirements:
CPU: 1.5GHz with 2+ cores
RAM: 4GiB
Storage: 32GiB
